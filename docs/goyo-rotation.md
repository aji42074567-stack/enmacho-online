# 今日の御用 — 日替わりローテ表と文面

作成: 2026-07-24 / 前提: daily-reward-design.md(仕様)・growth-design.md(報酬値)
ゲーム内表記は「クエスト看板」に統一(設計書内の「高札」は同じものを指す)

## 方式: 曜日固定ローテ

日付ハッシュではなく**曜日固定**にする。「水曜はアッキの日」が全プレイヤーの共通知識になり、
「水曜夜に集まろう」という約束が生まれる(チャットゲーム化の狙いに合致)。JST 0時切替。

## クエスト看板の文面(布告調・コピペ用)

閻魔庁の役所らしく「布告」の文体で統一する。形式:【御用】+件名+本文+報酬印。

- 月:【御用】ガキども騒がしき件
  「村の垣根にガキの群れが取り付いておる。十匹、成仏させて参れ。新入りの同行、大いに結構。」
- 火:【御用】イヌガミ徘徊の件
  「森の道にイヌガミが徘徊し、行き交う魂が難儀しておる。十匹の鎮めを命ず。」
- 水:【御用】アッキ乱心の件
  「森の奥のアッキが徒党を組んでおる。十匹、打ち鎮めよ。深追いは無用。」
- 木:【御用】骨鳴りの件
  「ガシャドクロの骨鳴りが夜毎うるさいとの訴えあり。十体、静めて参れ。」
- 金:【御用】洞窟二層 闇祓いの件
  「試練の洞窟二層に闇が濃い。ヤミイヌガミ・オニビドクロ合わせて十、祓うべし。連れ立って行け。」
- 土:【御用】ゴズ討伐の件
  「洞窟深部のゴズは一人手に余る。仲間を募りて十体を鎮めよ。庁は同行を推奨する。」
- 日:【御用】無縁塚 骨鎮めの件
  「無縁塚のシラホネ十体の供養を命ず。遠路である。道中の魂にも合掌を忘れるな。」

文面はUIの1〜2行に収まる長さ。達成時の納品文は共通:
「御用、しかと確かめた。本日の勤め、大儀である。」(+報酬表示)

## 実装メモ

- 曜日配列で持つ: `GOYO_ROTATION = { field: [月...日], rinne: [月...日] }`(対象mob型IDの配列)
- 金曜の鵺のような時間帯限定対象は、対象が出ない時間帯でも受注可・進行は出現時間のみ
- 文面はQUESTS同様の定義配列に。将来章が増えたら表を足すだけ
- 計測: 既存の quest_accept/quest_complete に quest_id=`goyo_{曜}` で乗る(新イベント不要)
