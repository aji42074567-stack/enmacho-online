# デイリークエストと入庁恩賞の設計(改訂版・功徳札経済)

作成: 2026-07-24(旧版を全面置換) / 前提: chapter-gear-design.md(功徳札・交換所)・
boss-material-design.md(竜の間)・growth-design.md(時給基準)

## 日課の三段構え

| 段 | 必要なもの | 主な報酬 |
|---|---|---|
| 入庁恩賞 | **開くだけ** | 功徳札1+7日周期の品(時給5%相当) |
| 今日の御用 | 遊ぶ(ソロ可・PT優遇) | 時給25%のモン/XP+功徳札2 |
| 竜の間 | PT2名以上・1日3回 | 竜の血 確定1/回 |

軽い方から「開く→遊ぶ→誘う」へ持ち上げる。報酬の軸は**功徳札**に一本化し、
使い道はすべて交換所(chapter-gear-design.md §4)に集める。

## 1. 入庁恩賞(ログイン報酬)

毎日: **功徳札1枚**+7日周期の品。連続が切れても周期が1日目に戻るだけ(責めない)。

| 日 | 品 |
|---|---|
| 1 | 甘露水(帯の標準サイズ)×3 |
| 2 | 功徳札+2 |
| 3 | 巻物のかけら×2 |
| 4 | 韋駄天の水×1 |
| 5 | 功徳札+3 |
| 6 | 巻物のかけら×3 |
| 7 | **竜の血×1** |

- 薬の配布量は所持上限(甘露水50・バフ薬20)に対して十分小さく、溢れ問題は起きない。
  受取時に上限超過分は功徳札1に自動変換(捨てさせない)
- 住民(定住許可証)は品2倍+札2枚/日(monetization-plan.mdの恩賞2倍をこの形で確定)

## 2. 今日の御用(デイリークエスト)

- 高札で受注。**その章の看板妖怪から日替わりで1体**が対象(JST 0時更新・全員共通)。
  「今日は土蜘蛛の日」= みんな同じ場所に集まる→自然にPTが湧く。討伐数は10体(眷属含む)
- 報酬(裁定・2026-07-25改訂): **「自分のレベルの敵25体ぶん」で計算**+**功徳札2枚(固定)**
  - XP = 敵XP目安(自Lv)×25、モン = 敵ゴールド目安(自Lv)×25(式は徳位表シート)
  - 意味は「約15分の狩りぶん」。帯中央のプレイヤーでは従来の「時給25%」と一致する
  - 帯中央固定だと序盤の新規が一気に数レベル跳び、オンボーディングの難度曲線を
    追い越してしまうため(実測: 徳位1→8)、自Lvスケールに変更した
- チーム共有カウント+チーム達成で報酬1.3倍+募集ポップアップ(chatgame-design.md §2)は従来どおり
- 看板妖怪ローテにすることで、御用がそのまま**職別レアドロを狙う口実**になる
  (御用こなしながら3%を引く→外れ職でも分解で札5枚→交換に近づく。全部つながる)

## 3. 功徳札の収支設計(1日あたり)

| 入手 | 札/日 |
|---|---|
| 恩賞(毎日分+周期日平均) | 約1.7 |
| 御用 | 2 |
| レアドロ分解(狩り1〜2時間・期待値) | 0〜5 |
| 合計(毎日遊ぶ人) | **約4〜9枚** |

| 使い道 | 札 | 到達日数の目安 |
|---|---|---|
| 職指定レアドロ交換 | 20 | 3〜5日(ドロ運の天井として機能) |
| 月替わり見た目装備 | 30 | 月の前半で届く。皆勤不要 |
| 巻物のかけら(上限なし・2026-07-25撤廃) | 2/個 | 補助的な強化燃料。価格自体がブレーキ |

ログインだけの人でも月50枚前後=月替わり装備+α に届く。**開くだけでも月1つの楽しみが確定**し、
遊ぶ人は職装備の天井が加速する——負担なしの二層構造。

## 4. 竜の間(変更なし・再掲)

血1確定×1日3回(PT2名以上)。血は移し打ち(+値の引っ越し)専用。
功徳札経済とは意図的に切り離す(血=装備資産の通貨、札=収集と天井の通貨)。

## 5. 実装メモ

- すべてクライアント完結(worker不要)。JST判定は`JST_OFFSET_MS`再利用
- セーブv16でまとめて追加: `daily`(御用)・`onsho`(恩賞)・`fuda`(札)・`dragonRoom`(竜の間回数)
- 計測: `onsho_claim`・`quest_accept/complete`(既存)・`fuda_exchange`(品目付き)・
  `dragon_room_clear`(人数付き)をGA4へ。日課ごとの継続率が個別に見える
- 旧版(装備のかけら・月替わり装備のログイン配布)は廃止。月替わり装備は交換所商品へ移動済み
