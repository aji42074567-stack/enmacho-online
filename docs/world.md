# 世界観設計書 — 閻魔庁ONLINE

> 死後に知る。生前こそが地獄だった。

## 0. コンセプト(一行で)

「死後の世界を管理する巨大役所」だと思っていたら、
実は現実世界そのものが魂を育成・選別する仮想現実だった——という二層構造のRPG。
遊びながら、プレイヤー自身が自分の人生を振り返ってしまう仕掛けを随所に置く。

## 1. 二層構造

### 表の世界観(プレイヤーが最初に信じている設定)

人は死ぬと**地獄庁**へ送られ、人生を審査される。
罪を償い、徳を積み、**転生許可証**を得れば再び現世へ戻れる。
組織の正式名称は「地獄庁」。トップが閻魔大王なので、亡者たちは通称**「閻魔庁」**と呼ぶ。

### 裏の真実(終盤まで隠す)

- プレイヤーが「現世」だと思っていた人生そのものが仮想世界
- 地獄庁は死後の世界ではなく、仮想現実を管理する**システム管理領域**
- ゲームの舞台は、死後の世界に見せかけた**巨大サーバーの管理画面**

## 3. 真実の明かし方 — 三幕構成

**第一幕「信じる」(v0.3〜)**
プレイヤーは地獄庁の新入り亡者として素直に徳を積む。
ただしログに稀に(4%程度)役所らしからぬ「系統メッセージ」が混ざる。
例:「[系統] 徳パラメータ更新…記録完了」
NPCも時々ぽろっとこぼす。「この世界、時々"ほころび"が見える」

**第二幕「違和感」(クエスト実装後)**
クエストを進めると矛盾に出会う。
記憶がループしている亡者、存在しないはずの部屋、
閻魔大王が一度も肉声で話さないこと。

**第三幕「真実」(ストーリー終盤)**
管理画面領域に踏み込み、真実を知る。
そのうえで「それでも転生するか?」という選択をプレイヤーに投げる。
※ 明かした後が一番面白くなるように、真実を知ってからの周回要素を用意する。

## 4. 教訓メッセージの出しどころ(説教にしない)

原則: **短く、感情が動いた瞬間に、一言だけ。**

1. **死亡画面の「審査官コメント」** — 死ぬたびに一言。
   例:「審査記録: 無謀は勇気ではない。だが、挑んだ事実は徳として記録した」
2. **懸衣翁の会話** — 攻略ヒントに人生の話を混ぜる。
   例:「六文銭は使ってこそ意味がある。生前、金を抱えたまま死んだ者を何人も見た」
3. **戦闘の意味づけ** — 敵を倒す=殺生ではなく「迷いを断って成仏させる功徳」。
   数字も「討伐数」ではなく「成仏させた魂」と呼ぶ
4. **アイテム説明文**(将来) — 装備の一つひとつに元の持ち主の未練を書く
5. **レベルアップ** — 「レベル」ではなく「徳位」。強さ=人格の成熟という建て付け

## 9. 第一章「賽の森」

賽の森は、転生審査へ進む魂を見極める最初の島。
怪物は生前の強欲・怒りだけでなく、悲しみ・孤独・恐れ・執着を抱えきれず、
人の姿を保てなくなった魂の成れの果てでもある。

プレイヤーの戦闘は魂を消滅させる討伐ではない。
まとわりついた業を断って魂を浄化し、人の姿と輪廻転生の機会を返す仕事である。
これは「因果応報」を永遠の処罰ではなく、自ら蒔いた種を刈った後に
もう一度選び直せる仕組みとして描くための第一章の核になる。

カズーは記憶処理が不完全な元渡し守。酔客を装いながら、段階的に島の役割を語る。
最後の門番ゴウリュウも、喪失への怒りを手放せなかった元人間である。
鎮魂後は海霧が晴れ、渡し船と「渡し守の六文銭」によって第二章への道が開く。
六文銭は船賃として消費する金ではなく、元渡し守カズーから託される、
次の審査区へ渡る資格の証である。

## 10. 第二章「転生編」構想

海の向こうの広大な審査区へ渡る前に、魂の役目を四つから選ぶ。

| 職 | 役割 |
|---|---|
| 羅刹 | 力と肉体を活かす近接戦闘 |
| 影法師 | 速さ・会心・間合いを活かす遊撃 |
| 呪禁師 | 知力・術・状態異常を扱う制圧 |
| 護法僧 | 防御・回復・支援を担う守護 |

第一章で積んだ徳位は失わせず、職ごとの成長を新たに重ねる。
