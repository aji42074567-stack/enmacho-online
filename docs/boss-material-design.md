# 竜の血と「竜の間」の設計(改訂版)

作成: 2026-07-24 / 前提: chatgame-design.md(新強化・PT募集)・growth-design.md(8章構成)
決定: **竜の血=+値を移す素材。ゴウリュウ=PTで毎日通うデイリーボスにする**

---

## 1. 竜の血 = 「移し打ち」(+値の引っ越し素材)

### 仕組み

鍛冶屋カナヤマの新メニュー「移し打ち」:
**装備Aの+値を装備Bへ移す。移動元は+0に戻る。失敗なし(確定成功)。**

- 費用: **竜の血3+手数料(その帯の時給モンの50%)**
- 同種間のみ: 武器→武器、防具→防具(胴→胴など同部位)
- 移動先に+値が付いている場合は上書き(高い方が残る、ではなく完全上書き。単純に)

### なぜこれが効くか

新強化仕様では+値は失われない資産になる。すると章が進むたび
「育てた+12の打刀を、新しい武器のために捨てられない」問題が必ず起きる。
移し打ちがあれば**強化資産ごと新武器へ乗り換えられる**ので、

- 新章・新武器が出るたびに竜の血の需要が確実に跳ねる(恒久的な需要サイクル)
- 「とりあえず今の武器を強化する」ことが無駄にならない → 強化を今すぐ楽しめる
- ボス武器を引いた瞬間が「移し打ちだ!」という最高の瞬間になる

## 2. デイリー専用「竜の間」

### 仕様

- 竜の洞窟の最深部に専用の入口「竜の間」を追加
- **2名以上のチーム必須**。ソロでは扉が開かない(閻魔庁の掟として世界観で説明)
- **1人1日3回まで**(JST 0時リセット。回数はセーブに記録)
- 中にいるのは竜の間専用のゴウリュウ(調整版)。倒すと退出、また入り直せる

### 竜の間ゴウリュウの調整(フィールド版とは別個体)

| 項目 | フィールド版(現行) | 竜の間版 |
|---|---|---|
| HP | 13,600 | 6,000+人数×2,000(2人=10,000 / 4人=14,000) |
| 攻撃 | 80-180 | 50-110(2章帯が薬を飲みながら受けられる上限) |
| 復活 | 討伐30分後 | 入室ごとに出現 |
| ドロップ | 業龍断ち等レア+血7% | **竜の血1確定**+モン+XP(帯の時給準拠)。レア装備・巻物なし |

- レア装備と巻物はフィールド版の専売のまま → 竜の間は「血の安定供給所」に役割を限定
- 1日3回×血1確定=最大3血/日 → **毎日通えばちょうど1日1回の移し打ちペース**(費用3血と対応)
- フィールド版ゴウリュウの存在価値(30分復活・レア狙い)は食わない

### ソーシャル効果(これが本命)

- 「2名以上必須+毎日3回+全員に血が確定」= **毎日必ず発生する『誘う理由』**
- 募集ポップアップ(chatgame-design.md §2)の対象に竜の間を含める:
  「◯◯さんが竜の間への同行者を募っています〔加勢する〕」— 本日の残り回数がある人にだけ表示
- デイリーの御用と並ぶ日課の二本柱: 御用(軽い・ソロ可)と竜の間(PT必須・確定報酬)

## 3. 強化の血要求は廃止(裁定・2026-07-24)

- **移し打ち導入と同時に、武器強化+7以降の竜の血要求を廃止する。** 血は移し打ち専用に一本化
  (1素材1役割)。移行までの現行仕様は暫定として維持してよい
- 章ボスの永続価値は「器開き」で担保: +10・+13・+16の壁を初めて超えるとき、
  装備1点につき章ボス素材1個(**1回だけ。挑戦ごとではない**)。挑戦ごとの素材要求は
  低成功率帯(+16以降1%)で二重の壁になるためやらない

## 4. 復活アイテムは道具屋の高額商品に(竜の血と切り離す)

戦闘中に倒れた仲間を起こす「甦りの香」を道具屋で販売。素材不要・モンのみ。
価格は**その帯の時給モンの50%**(1章帯 約2,700モン、2章帯 約6,500モン)。

- 高額なので常備はためらう=護法僧の奥義「甦生」(徳位45)の価値は保たれる
- モンの立派な使い道(シンク)が一つ増える

## 5. 実装メモ(Claude Code向け)

- **竜の間はチーム単位の部屋が必要**(他チームと共有しない)。共有ワールドWorkerに
  チームIDつきの動的ゾーン(例: `dgroom-{partyId}`)を追加する形になるため、
  AGENTS.mdの「ゾーン追加4箇所」ルールの拡張+`deploy_worker.sh`が必要な、
  これまでで一番Worker寄りのタスク。実装順ロードマップでは職スキル第1弾の後を推奨
- 挑戦回数はセーブへ `dragonRoom:{date,count}` として記録(JST基準は既存のJST_OFFSET_MSを再利用)
- 移し打ちはクライアント完結(カナヤマのUIとセーブ処理のみ)なので先行実装できる。
  こちらは軽く、竜の間より前に出してよい(血の在庫はフィールド版7%で貯まり始める)
- balance-model.xlsx「強化新案」の週間巻物入手目安は現行のまま。血の供給が変わっても
  巻物供給は変わらない設計(竜の間は巻物を落とさない)

## 6. 章ボス素材の一般則(2章以降)

竜の血の役割が「移し打ち」で確定したので、以後の章ボス素材はシンプルに:

- **各章の頂点ボス素材=その章から始まる新しい仕組みの鍵**を1つずつ担当させる
  (例: 2章・酒呑童子の素材=「器開き」の要求素材、3章素材=装備覚醒、4章素材=式神…)
- 1素材1役割。竜の血のように役割が明確なら、プレイヤーは「何のために狩るか」で迷わない
- ペット/式神は竜の血から切り離したので、将来の章の目玉として温存できる
