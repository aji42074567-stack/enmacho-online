# 章別モンスターと職別レアドロ装備の設計

作成: 2026-07-24 / 前提: growth-design.md(8章)・chatgame-design.md(4職)・boss-material-design.md(竜の間)
決定: **装備は敵の名を背負って落ちる。章ごとに土地の妖怪+職別レアドロ+交換所、で1セット**

---

## 1. 章の作り方テンプレ(モンスター→装備→交換所)

各章はこの3点セットで量産する:

1. **土地の妖怪**: 章の地方にゆかりの妖怪を区ごとに1体(看板妖怪)+色替え眷属の雑魚
   (1章の鬼・龍・骸骨=死国よみの国、と同じ発想を各地方で繰り返す)
2. **職別レアドロ**: 看板妖怪は**その妖怪の名を冠した職別装備**(羅刹/影法師/呪禁師/護法僧)を
   ドロップ。既存ルールの運込み上限3%(MAX_RARE_EQUIPMENT_DROP_RATE)を踏襲
3. **交換所**: 章の町に「交換所」。職違いのレアドロを分解して札にし、自職の装備と交換できる

ステータスは balance-model.xlsx の式から(店売り=帯中央Lv×1.2、レアドロ=その1.15〜1.3倍)。
名前と見た目だけが章ごとの個性で、数値は式で機械的に決まる——絵に集中できる構造。

## 2. 二章・近畿の妖怪案(区の看板)

| 区(候補) | 妖怪 | 由来 | 役どころ |
|---|---|---|---|
| 滋賀 | 大百足(おおむかで) | 三上山・俵藤太伝説 | 琵琶湖岸の看板。長い体で多段当たり |
| 京都 | 鵺(ぬえ) | 平安京・夜の御所 | 夜時間のみ出現(昼夜サイクル活用) |
| 大阪 | 茨木童子 | 茨木・鬼伝説 | 中ボス。羅刹的な剛腕 |
| 奈良 | 土蜘蛛 | 葛城山 | 巣で足止め攻撃(影法師の影縫いと対) |
| 和歌山 | 清姫 | 道成寺・蛇体 | 執念の追跡型(aggro長い) |
| 三重 | 一目連(いちもくれん) | 多度・一つ目の竜神 | 風の遠隔攻撃 |
| 兵庫 | 件(くだん) | 西宮・予言する牛 | 戦わないNPC寄り。予言=ヒント役も面白い |
| 章ボス | **酒呑童子** | 大江山の鬼の頭領 | 二章の頂点。1章の鬼の「王」として物語も接続 |

7件あるので6区構成なら1つ削るか、件をNPCに回す。眷属雑魚は各看板の色替え
(仔蜘蛛・小鬼・蛇衆など)でスプライトを節約できる。

## 3. 遠距離は呪禁師の専売(職の個性として)

遠距離戦闘は**呪禁師だけ**に持たせる。武器のカテゴリではなく、職の特権にする:

- **呪禁師の通常攻撃は射程3マス**(武器は呪符・法具系。飛翔する札の弾を描画)
- バランスの機械則: 呪禁師の武器攻撃力は同格の他職の**0.85倍**(安全に殴れる代償。
  式はbalance-model.xlsxの武器式×0.85)
- 他職は近接のまま: 羅刹=大太刀、影法師=双刀(遠距離の一手はスキルの影針が担当)、
  護法僧=錫杖。「遠くから安全に」は呪禁師を選ぶ理由そのものにする
- 弾薬・消耗の概念は入れない(荷物系の面倒は作らない)
- 実装は「呪禁師の通常攻撃だけ射程あり」に限定されるので軽い。敵側の遠隔攻撃
  (一目連の風など)は既存のボスブレス処理の流用で足りる

## 4. 章あたりの装備構成(量産の定数)

| 区分 | 点数/章 | 中身 |
|---|---|---|
| 店売り武器 | 4 | 各職1本(呪禁師のみ遠距離型)。性能は式から自動 |
| レア武器 | 4 | 各職1本・章ボス産(章の頂点の記念品) |
| 店売り防具 | 4部位 | 職共通デザイン(職色違いはtintで無料) |
| レア防具 | 8 | **章ごとに2部位ローテ**(2章=胴・頭、3章=腕・足…)×4職。8章で全部位2周 |
| 計 | **約20点/章** | 1章の実装済み装備量とほぼ同規模=持続可能なペース |

やらないことも決めておく: レア防具を毎章4部位×4職にすると16点になり、交換所の
職指定天井(札20)がぼやける。**部位ローテで「今章は胴と頭の番」という個性を作る**。

## 5. 職別レアドロの命名と割り付け

命名規則: **「妖怪名+部位/得物」**。例:

- 羅刹: 酒呑童子の宴刀 / 茨木童子の鬼腕籠手 / 大百足の節鎧
- 影法師: 鵺鳴りの双刀 / 土蜘蛛の糸繰り足袋 / 清姫の追い帯
- 呪禁師: 一目連の風詠み呪符(遠距離) / 清姫の蛇鱗呪具 / 鵺の夜声の呪冠
- 護法僧: 道成寺の鐘錫杖 / 件の告げ法衣 / 大百足退治の護り笠

割り付けルール(悩まないための機械則):

- レア武器4本は**章ボス産**(職別4点から等確率1点・3%上限)。武器は章の頂点の記念品にする
- レア防具(2部位×4職=8点)は**看板妖怪に分散**(1体につき職別4点=1部位ぶん、を2体に)
- 雑魚(眷属)は職共通の小物1点のみ(帯・履物など)。レアの主役は看板と王に集中させる

**職違いが出るのは仕様**(4分の3は他職の装備)。それが交換所と、後述のPT・フレンドへの
「これ◯◯さんの職のじゃない?」という贈り物会話を生む。social.jsのギフト機能がここで生きる。

## 6. 交換所と「功徳札(くどくふだ)」

章の町の交換所NPCで(職違いレアドロは武器・防具とも対象):

- **分解**: 職違いレアドロ1点 → 功徳札5枚
- **交換**: 功徳札20枚 → その章の職指定レアドロ1点(自職を選べる)
  (=職違い4点で自職1点。ドロップ運の悪さに天井を作る)
- **月替わり見た目装備**: 札30枚(性能は店売り同等・見た目限定。絵はあじさん製。
  恩賞の「装備のかけら」案はこれに置き換えて廃止)
- 巻物のかけら: 札2枚/個・**月30個まで**(巻物供給の設計値を壊さない購入上限)

功徳札はデイリー(御用・恩賞)からも供給する(daily-reward-design.md改訂版を参照)。
**狩る人はレアドロ分解で、毎日来る人はデイリーで、同じ札に合流する**一本道の経済にする。

## 7. 所持制限の方針(荷物の重さは採用しない)

- 重量・所持枠の圧迫は**採用しない**(嫌いなものは入れない。正しい判断)
- 例外として**回復系のみ所持数上限**: 甘露水 各サイズ50、韋駄天・修羅 各20、甦りの香 5
  (無限ポーションによるボスのゴリ押し防止。それ以外の目的はない)
- **住民(定住許可証)は倉庫を開放**: 装備・アイテムを預けられる保管庫(200枠)。
  持ち歩き上限は無料と同じ(戦闘力は平等)、増えるのは保管だけ。
  monetization-plan.mdの「倉庫拡張」項をこの仕様で確定とする

## 8. 実装メモ

- 交換所・分解・札はクライアント完結(セーブに `fuda` を追加。v16枠に同乗)
- 職別レアドロはRARE_DROPSの既存構造そのままに、job属性を1つ足すだけで載る
- 妖怪の当たり判定・攻撃パターンは既存敵のパラメータ違いから始め、
  固有ギミック(鵺の夜限定・清姫の長追跡など)は1体1ギミックまでに抑える(作りすぎない)
