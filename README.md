# 閻魔庁ONLINE

> 死後に知る。生前こそが地獄だった。

死後の世界を管理する巨大役所「地獄庁」を舞台にした、
クォータービュー + クリック移動のブラウザRPG。
迷える魂を成仏させて徳を積み、転生許可の審査を目指す。

世界観の全体像(とこのゲーム最大の秘密)は [docs/world.md](docs/world.md) を参照。

## 遊び方

- **地面をクリック/タップ** … 移動
- **迷える魂(モンスター)をクリック** … 自動攻撃(成仏)
- **奪衣婆・懸衣翁をクリック** … 会話・買い物
- **E** 破魔の印 / **R** 回向(徳位3〜) / **F** 業火(徳位6〜) / **Q** 甘露水 / **I** 装備
- 目標: 賽の森の最深部に巣くう **業龍** を鎮める

進行状況(徳位・六文銭・装備)はブラウザに自動セーブされます。

## 技術メモ

- `index.html` 1ファイル完結。ライブラリなし(素のJavaScript + Canvas)
- サーバー不要。ファイルをダブルクリックするだけでも動く
- 公開は GitHub Pages(mainブランチにpushすると自動反映)
