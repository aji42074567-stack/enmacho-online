# アデンの森 — リネージュ風ブラウザRPG

リネージュ1のような「クォータービュー + クリック移動 + 狩りとレベル上げ」を
ブラウザだけで遊べるようにした自作ミニRPG。

## 遊び方

- **地面をクリック/タップ** … 移動
- **モンスターをクリック** … 自動攻撃
- **村人をクリック** … 会話・買い物
- **E** エナジーボルト / **R** ヒール(Lv3〜) / **F** ファイアボール(Lv6〜) / **Q** ポーション / **I** 装備
- 目標: 森の最深部に棲む **ドレイク** の討伐

進行状況(レベル・お金・装備)はブラウザに自動セーブされます。

## 技術メモ

- `index.html` 1ファイル完結。ライブラリなし(素のJavaScript + Canvas)
- サーバー不要。ファイルをダブルクリックするだけでも動く
- 公開は GitHub Pages(mainブランチにpushすると自動反映)
