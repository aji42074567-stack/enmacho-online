# 閻魔庁ONLINE

> 死後に知る。生前こそが地獄だった。

死後の世界を管理する巨大役所「地獄庁」を舞台にした、
クォータービュー + クリック移動のブラウザRPG。
迷える魂を浄化して徳を積み、転生許可の審査を目指す。
公開中の第一章「賽の森」では、カズーの因果話を追いながらゴウリュウを鎮め、
海の向こうへ渡る資格を得るところまで遊べる。

世界観の全体像(とこのゲーム最大の秘密)は [docs/world.md](docs/world.md) を参照。

## 遊び方

- **地面をクリック/タップ** … 移動
- **迷える魂(モンスター)をクリック** … 自動攻撃(成仏)
- **町のNPCをクリック** … 会話・買い物・鍛冶
- **E** 破魔の印 / **R** 回向(徳位3〜) / **F** 業火(徳位6〜) / **Q** 選択中の甘露水 / **I** 装備
- **⚙** … BGM・効果音・明るさ・画面揺れの設定
- 目標: 高札の「酔客の因果話」を進め、竜の洞窟の **ゴウリュウ(業龍)** を鎮める

進行状況(徳位・モン・装備)はブラウザに自動セーブされます。
魂の資質ポイントは初期5ポイントに加え、徳位が上がるたびに1ポイント得られます。
資質は「力・体・速・知・運」の5種類で、速は会心と回避、知はMP量とMP回復、
運はドロップ率に影響します。

## 技術メモ

- `index.html` + `assets/`(画像)。ライブラリなし(素のJavaScript + Canvas)
- サーバー不要。フォルダごと置けばダブルクリックだけでも動く
- 画像が読み込めない環境では従来のコード描画に自動フォールバック
- 森床・土道・石畳・建物・樹木・岩・村の小物は画像素材で描画
- 主人公は男女とも4方向・最大8コマの滑らかな歩行アニメーションに対応
- 公開は GitHub Pages(mainブランチにpushすると自動反映)
